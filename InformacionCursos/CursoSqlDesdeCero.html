Curso de SQL desde 0

Este curso introductorio te enseñará los conceptos básicos del lenguaje SQL, el estándar para gestionar bases de datos relacionales. Aprende a realizar consultas, modificar tablas y gestionar la información de forma eficiente.

Comenzar

Contenido del Curso

5 Secciones - 30 Lecciones - 6h en total

  • Introducción a Bases de Datos y SQL - 45 min
  • Consultas SELECT: Extrayendo Datos - 1h 15 min
  • Modificación de Datos con INSERT, UPDATE y DELETE - 1h
  • Gestión de Tablas y Estructuras de Bases de Datos - 1h 15 min
  • Proyecto Práctico: Creación de una Base de Datos Completa - 1h 45 min

Habilidades que obtendrás

  • Comprender los conceptos fundamentales de SQL
  • Realizar consultas a bases de datos
  • Modificar y gestionar tablas y datos
  • Diseñar y crear bases de datos relacionales
  • Gestionar proyectos con bases de datos desde cero

Requisitos Mínimos

No se requieren conocimientos previos. Este curso es ideal para principiantes que desean aprender a gestionar bases de datos con SQL.
